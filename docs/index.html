scalable

extensible

basic example

Here’s a Bayesian linear regression model for the iris data using greta:

x <- iris$Petal.Length
y <- iris$Sepal.Length
library(greta)

int <- normal(0, 5)
coef <- normal(0, 3)
sd <- lognormal(0, 3)

mean <- int + coef * x
distribution(y) <- normal(mean, sd)
m <- model(int, coef, sd)
draws <- mcmc(m, n_samples = 1000, chains = 3)
bayesplot::mcmc_trace(draws)
